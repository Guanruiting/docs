           <h1>.stop</h1>
           <!-- 
           点击目标事件，先走捕获->目标->冒泡 即捕获：从外到内依次执行
           冒泡：从内到外依次执行click事件
           -->
           <!-- 如下点击内部点击，阻止了冒泡过程，即只执行methodB这个方法，如果不加.stop，讲先执行方法，
               后执行methodA方法。即通过了冒泡这个过程。 -->
           <div v-on:click="methodA" class="greed h100">外部点击
               <div v-on:click.stop="methodB" class="red h50">内部点击</div>
           </div>


           <h1>.prevent</h1>
           <!-- 阻止了a标签的默认刷新  -->
           <a href v-on:click.prevent>prevent阻止刷新</a>


           <h1>.self</h1>
           <!-- 如果点击内部点击，冒泡不会执行methodA方法，
           因为event.target指的是内部点击的dom元素，不是外部点击的，所以不会触发自己的点击事件 -->
           <div v-on:click.self="methodA" class="greed h100">外部点击
               <div v-on:click="methodB" class="red h50">内部点击</div>
           </div>

           <h1>.capture</h1>
           <!-- 在点击最里层的最内点击时，methodA方法先执行，因为methodA方法在捕获模式执行的，先与冒泡事件。
           下列执行顺序 methodA->methodC->methodB ,因为后俩个还是冒泡模式下触发的事件。 -->
           <div v-on:click.capture="methodA" class="greed h150">外部点击
               <div v-on:click="methodB" class="red h100">内部点击
                   <div v-on:click="methodC" class="greed h50">最内点击</div>
               </div>
           </div>

           <h1>.once</h1>
           <!-- 只触发一次  -->
           <div v-on:click.once="methodB" class="greed h50">once</div>

    # 事件委托/代理

    <p>JS添加到页面中的事件处理程序数量直接关系到页面运行性能。与DOM不断进行交互，访问DOM的次数越多，引起浏览器重绘与重排的次数越多，延长整个页面的交互、就绪时间，因此性能优化的主要思想之一就是：减少DOM操作。</p><p>事件委托就是父级代为执行事件，子元素通过冒泡触发事件处理程序。</p>

    <!-- 使用事件代理 -->
<ul @click="change($event)">
 <li v-for="item in menus" :mark="item.mark" class="myLi">{{item.des}}</li>
</ul>

<!-- 没有使用事件代理 -->
<ul>
 <li v-for="item in menus" :mark="item.mark" class="myLi" @click="change($event)">{{item.des}}</li>
</ul>

    <p>代码实例</p>

    


   <meta charset="UTF-8">
   <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
   <title>1.1.3-Vue事件绑定-事件代理 事件委托</title>
   <style scoped>
       .left li{width: 100%;}
       .allWrap{display: flex}

    </style>
   



   <script type="text/javascript">
       window.onload = function() {
           vm = new Vue({
               el: '#app',
               data: {
                   menus:[
                       {mark:'one', des:'玄幻'},
                       {mark:'two', des:'言情'},
                       {mark:'three', des:'都市'},
                       {mark:'four', des:'科技'}
                   ]
               },
               methods: {

                   change: function(e) {
                       //当子元素比较多的时候
                       if (e.target.className === 'myLi') { // 判断仅当点击类名为‘'myLi'’的时候
                           console.log(e.target.innerHTML);
                       }

                     
                   }
               }

           });
       }

   </script> 

<div id="app" class="container">

   <div class="allWrap">
       <!-- 使用事件代理 -->
       <ul @click="change($event)">
           <li v-for="item in menus" :mark="item.mark" class="myLi">{{item.des}}</li>
       </ul>
       
       <!-- 没有使用事件代理 -->
       <ul>
           <li v-for="item in menus" :mark="item.mark" class="myLi" @click="change($event)">{{item.des}}</li>
       </ul>

   </div>

</div>
